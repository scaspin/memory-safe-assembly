Return from subroutine branches unconditionally to an address in a register, with a hint that this is a subroutine return.

RET {<Xn>}

integer n = UInt(Rn);

Assembler Symbols

<Xn>

Is the 64-bit name of the general-purpose register holding the address to be branched to, encoded in the "Rn" field. Defaults to X30 if absent.

Operation

bits(64) target = X[n, 64]; if (IsFeatureImplemented(FEAT_GCS) && GCSPCREnabled(PSTATE.EL)) then (PSTATE.EL) && n != 16) then target = LoadCheckGCSRecord(target, GCSInstType_PRET); SetCurrentGCSPointer(GetCurrentGCSPointer() + 8); // Value in BTypeNext will be used to set PSTATE.BTYPE BTypeNext = '00'; BranchTo(target, BranchType_RET, FALSE);
